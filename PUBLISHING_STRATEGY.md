# 📋 블로그 글 발행 전략: 현실적인 개수 및 일정

## 🎯 Tier별 발행 가능 개수

### Tier 1: Primary (주요 - 가장 현실적)

**시스템 설정**:
- 대상 종목: 13개
- 필요 기준: 높은 중요도 뉴스 3개 이상
- 글 길이: 2,500-3,500자 (SEO 최적화)

**월간 발행 가능 개수**:

```
시나리오 1: 주 1회 생성 (가장 현실적)
└─ 13개 종목 × 4주 = 52개 글/월
   └─ 글당 작성시간: 10분
   └─ 월 총 작성시간: 520분 (8.7시간)
   └─ 추천: 일주일에 한 번, 오후에 2-3시간 투자

시나리오 2: 주 2회 생성 (적극적)
└─ 13개 종목 × 8회 (월 2회 × 4주) = 104개 글/월
   └─ 월 총 작성시간: 1,040분 (17.3시간)
   └─ 추천: 월-수-금 또는 화-목에 각 2시간

시나리오 3: 일일 생성 (매우 적극적 - 비추천)
└─ 13개 종목 × 매일 = 380개 글/월
   └─ 월 총 작성시간: 3,800분 (63시간)
   └─ 현실성: 매우 낮음 (일일 7-8시간 투자 필요)
```

**추천 운영 방식**:
```
월요일: 2-3개 글 (30분 자동 + 20-30분 작성)
수요일: 2-3개 글
금요일: 2-3개 글

월간: 6-9개 글 (현실적이고 지속 가능)
연간: 72-108개 글
```

---

### Tier 2: Secondary (심화 분석)

**시스템 설정**:
- 대상 종목: 18개 (Tier 1 + 5개 추가)
- 필요 기준: 뉴스 3개 이상 AND 점수 75점 이상
- 글 길이: 2,500-3,500자

**월간 발행 가능 개수**:

```
시나리오 1: 주 1회 (Tier 1과 분리)
└─ 18개 × 4주 = 72개 글/월
   └─ 월 총 시간: 720분 (12시간)

시나리오 2: Tier 1과 병행 (주 2회)
└─ 13개 Tier 1 + 5개 Tier 2 = 18개 × 4주 = 72개 글/월
   └─ 월 총 시간: 720분 (12시간)

시나리오 3: 월 1-2회만 (최소 버전)
└─ 18개 × 1-2회 = 18-36개 글/월
   └─ 월 총 시간: 180-360분 (3-6시간)
```

**추천 운영 방식** (Tier 1 이후):
```
1-2개월 뒤 추가 시작
화-목에 Tier 2 글 2-3개씩 작성
월 20-30개 발행 (Tier 1과 합쳐서 월 30-40개)
```

---

### Tier 3: Trend (트렌드 리포트)

**시스템 설정**:
- 대상 종목: 15개 (동적 선택)
- 필요 기준: 월간 상위 15개 종목
- 글 길이: 3,000-4,000자 (종합 분석)
- 글 형식: 월간 종합 리포트 또는 개별 글

**월간 발행 가능 개수**:

```
옵션 1: 월 1회 종합 리포트
└─ 글 1개 (15개 종목 모두 포함) = 1개 글/월
   └─ 작성시간: 60-90분
   └─ 형식: "2025년 11월 최상위 주식 15개 분석"

옵션 2: 월 4회 개별 발행
└─ 15개 종목 ÷ 4주 = 3-4개 종목/주 = 12-16개 글/월
   └─ 작성시간: 월 120-160분 (2-3시간)

옵션 3: 월 2회 분할 리포트
└─ 상위 8개 + 하위 7개 = 2개 글/월
   └─ 작성시간: 월 120-150분
```

**추천 운영 방식**:
```
3개월 뒤 추가 시작
매월 1일: 월간 종합 리포트 발행 (1개)
또는 매 주 월요일: 해당 주 상위 4개 종목 분석

월 4-5개 글 추가
```

├─ 09:30-10:00: 뉴스 분석 (자동)
├─ 10:00-10:05: 프롬프트 생성 (자동)
└─ 13:00-15:00: Claude Code로 2-3개 글 작성 (수동, 120분)

수요일
└─ 13:00-15:00: Claude Code로 2-3개 글 작성

금요일
└─ 13:00-15:00: Claude Code로 2-3개 글 작성

월간: 6-9개 글
```

**월간 작업 시간**:
```

DB 저장: 5분/회 × 3회 = 15분

총: 약 7-10시간/월 (1시간 × 7-10일)
```

## 💪 실현 가능한 시나리오

### Conservative (보수적 - 추천)

```
목표: 월 10개 글
방식: 주 1회만 진행 (월요일 또는 목요일)
     - 3개월마다 재평가
     - 안정성과 품질 우선

시간: 월 2-3시간
지속성: ⭐⭐⭐⭐⭐ (매우 높음)
성과: 월간 12개 글 발행 (연 144개)
```

**구체적 일정**:
```
월요일
├─ 09:00-10:00: 뉴스 수집 + 분석 + 프롬프트 생성 (자동)
└─ 13:00-14:30: Claude Code로 3개 글 작성

매달: 월-목-월(다음 주) 반복
총: 월 12개
```

목표: 월 30개 글 (Tier 1 + 일부 Tier 2)

     - 월수금 또는 화목
     - 품질과 양의 균형

시간: 월 6-8시간 (주 1.5-2시간)
지속성: ⭐⭐⭐⭐ (높음)
성과: 월간 30개 글 발행 (연 360개)
```

**구체적 일정**:
```
월요일:  2-3개 글 (120분)
수요일:  2-3개 글 (120분)
금요일:  2-3개 글 (120분)

+매월 첫째 주: Tier 2 1-2개

월간: 30-35개
```

---

### Aggressive (공격적 - 풀타임 가능)

```
목표: 월 50+ 글 (Tier 1 + Tier 2 + Tier 3)
방식: 월-금 매일 진행
     - 최대 콘텐츠 생산

시간: 월 12-15시간 (주 2.5-3시간 × 5일)
지속성: ⭐⭐⭐ (중간)
성과: 월간 50-60개 글 발행 (연 600-720개)

**구체적 일정**:
```
월-금: 각 2-3개 글 (120분 × 5일 = 600분)
매주: Tier 2 추가 (상황에 따라)
매월: Tier 3 리포트 1개

월간: 50-60개
