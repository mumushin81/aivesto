# Discord 토큰 찾기 - 시각적 가이드

**한눈에 보는 스크린샷 가이드**

---

## 📍 **전체 과정 한눈에 보기**

```
1. Discord 웹 접속
   ↓
2. Chrome DevTools 열기 (Command+Option+I)
   ↓
3. Application 탭 클릭
   ↓
4. Cookies → https://discord.com 선택
   ↓
5. __Secure-user_token_r / _i 찾기
   ↓
6. Value 복사
```

## 🖼️ **토큰 형식 예시**

### TOKEN_R (Authorization Token)
```
┌─────────────────────────────────────────────────────────────┐
│ MTIzNDU2Nzg5MDEyMzQ1Njc4OTA.GaBcDe.FgHiJkLmNoPqRsTuVwXyZ    │
│ └────────┬────────┘ └──┬──┘ └───────────┬──────────┘        │
│        Part 1       Part 2          Part 3                   │
│                                                               │
│ • 길이: 약 70-90자                                           │
│ • 점(.)이 2개로 3부분 구분                                   │
│ • Base64 인코딩                                              │
└─────────────────────────────────────────────────────────────┘
```

### TOKEN_I (User ID Token)
```
┌─────────────────────────────────────────────────────────────┐
│ dXNlcl9pZF8xMjM0NTY3ODkwMTIzNDU2Nzg5MA%3D%3D                │
│ └────────────────┬──────────────────┘ └─┬─┘                 │
│            Base64 encoded          URL encoded              │
│                                                               │
│ • 길이: 약 40-60자                                           │
│ • %3D는 = 의 URL 인코딩                                      │
│ • Base64 + URL 인코딩 조합                                   │
└─────────────────────────────────────────────────────────────┘
```

## ✅ **성공 확인 체크리스트**

```
□ Discord 웹에 로그인됨
□ Chrome DevTools 열림 (Command+Option+I)
□ Application 탭 보임
□ Cookies → https://discord.com 선택됨
□ __Secure-user_token_r 찾음
□ __Secure-user_token_i 찾음
□ TOKEN_R 복사 완료 (약 70-90자)
□ TOKEN_I 복사 완료 (약 40-60자)
□ 앞뒤 공백 없음 확인
□ 따옴표 제외 확인
```

## 🎯 **다음 단계**

토큰을 찾았다면:

```
1. 메모장에 안전하게 저장
   TOKEN_R=MTIzNDU2Nzg5...
   TOKEN_I=dXNlcl9pZF8x...

2. Claude Desktop 설정
   $ nano ~/Library/Application\ Support/Claude/claude_desktop_config.json

3. 또는 자동 설정 스크립트
   $ ./scripts/setup_midjourney_mcp.sh
```

## 🆘 **여전히 안 될 때**

### 최종 체크리스트:
```
□ 일반 브라우저 창 (시크릿 모드 ✗)

→ Discord 로그아웃 → 재로그인 → 다시 시도
```
