## 🔍 Part 1: 뉴스 분석 시스템

### 1.1 분석 기준 (6가지 항목)

#### ① 관련성 점수 (Relevance Score)

```
100점 기준 시스템:

0-30점:   무관 (일반뉴스, 정치, 스포츠)
31-60점:  간접 관련 (경제 일반, 업계 트렌드)
61-80점:  직접 관련 (특정 기업/섹터 뉴스)
81-100점: 매우 중요 (실적, M&A, 규제, 중대 사건)

예시:
- Apple Q4 실적 발표: 95점 (직접 관련 + 매우 중요)
- Fed 기준금리 인상: 75점 (경제 지표 + 섹터 영향)
- 일반 경제뉴스: 50점 (배경지식만 필요)
```

#### ② 영향 대상 (Affected Symbols)

```
최대 5개 종목까지:
- 직접 언급된 기업만 포함
- 간접 영향 제외 (예: 공급업체X)
- 예: Apple 뉴스 → [AAPL] (또는 [AAPL, MSFT] if 경쟁 관련)
```

#### ③ 주가 영향 (Price Impact)

```
3가지 방향:

📈 UP (긍정적 영향)
  - 실적 증가
  - 신제품 출시
  - 호실적
  - 규제 완화
  예: MSFT AI 투자 → UP

📉 DOWN (부정적 영향)
  - 손실 발생
  - 소송
  - 규제 강화
  - 경쟁 심화
  예: Apple 아이폰 판매 급감 → DOWN

⚪ NEUTRAL (중립)
  - 장단점 혼재
  - 즉각적 영향 없음
  예: 경영진 교체 → NEUTRAL
```

#### ④ 중요도 (Importance)

```
3단계 분류:

🔴 HIGH (높음)
  - 즉각적 주가 영향 예상
  - 24시간 내 반응 기대
  예: 분기 실적 발표

🟡 MEDIUM (중간)
  - 중기 영향 (주 단위)
  예: 신제품 로드맵 발표

🟢 LOW (낮음)
  - 장기적/간접적 영향
  예: 업계 트렌드 변화
```

#### ⑤ 분석 근거 (Reasoning)

```
2-3문장 설명:
- 왜 이 점수인지
- 주가에 어떤 영향을 줄지
- 핵심 팩트

예:
"MSFT가 Azure에 $10B 추가 투자로,
AI 경쟁에서 우위 확보.
클라우드 수익성 개선으로 2026년부터
이익 증대 예상."
```

#### ⑥ 핵심 포인트 (Key Points)

```
3-5개 bullet points:

예:
• 2025년 클라우드 시장 16.4% 성장 예상
• MSFT Azure 점유율 23% (AWS 32% 대비)
• AI 통합으로 엔터프라이즈 고객 확대
• 경쟁사 대비 AI 기술 우위 확보
```

### 1.2 분석 프롬프트 생성 프로세스

#### 입력 데이터
```
{
  "news_id": "uuid",
  "title": "Apple Q4 earnings beat expectations",
  "content": "Apple reported record Q4 earnings...",
  "source": "Finnhub",
  "published_at": "2025-11-13T10:00:00Z",
  "symbols": ["AAPL"]
}
```

#### 생성되는 프롬프트 구조
```markdown
당신은 미국 주식 시장 전문 애널리스트입니다.

뉴스 제목: {title}

뉴스 내용:
{content[:2000]}

기존 추출된 심볼: {symbols}

다음 항목을 JSON 형식으로 분석:
1. relevance_score (0-100)
2. affected_symbols (최대 5개)
3. price_impact ("up", "down", "neutral")
4. importance ("high", "medium", "low")
5. reasoning (2-3문장)
6. key_points (3-5개 bullet)

응답 형식 (JSON만):
{ JSON 결과 }
```

#### 출력 형식
```json
{
  "news_id": "uuid",
  "relevance_score": 92,
  "affected_symbols": ["AAPL"],
  "price_impact": "up",
  "importance": "high",
  "reasoning": "Apple exceeded expectations by 15%, signaling strong demand...",
  "key_points": [
    "Q4 revenue: $123.5B (+10% YoY)",
    "iPhone sales surge 25%",
    "Services revenue hit record high"
  ]
}
```

### 1.3 신호 레벨 분류 (자동)

```
분석 결과 → 자동 레벨링:

Level 1: 🔴 URGENT (90+ 점)
  - 즉시 조치 필요
  - 대시보드 TOP 표시
  - 이메일 즉시 발송
  예: Apple Q4 실적 (점수 92, HIGH)

Level 2: 🟠 HIGH (80-89 점)
  - 고려 필요
  - 대시보드 우선 표시
  - 시간별 알림
  예: MSFT AI 투자 (점수 85, HIGH)

Level 3: 🟡 MEDIUM (70-79 점)
  - 모니터링
  - 대시보드 표시
  - 일일 요약
  예: 업계 트렌드 (점수 75, MEDIUM)

Level 4: 🟢 LOW (<70 점)
  - 참고용
  - 대시보드 하단
  - 주간 리포트
  예: 경제지표 (점수 65, LOW)
```

## 📝 Part 2: 블로그 글쓰기 시스템

### 2.1 글쓰기 기준 (11단계 필수 구성)

#### 1️⃣ 제목 (60자 이내)

```
목표: 검색 클릭률 극대화

약한 제목 예:
- "MSFT 최신 뉴스"
- "애플 분석"
- "마이크로소프트 관련 소식"

강한 제목 예:
- "MSFT 클라우드 전략 대전환, AI 투자 $10B 발표"
- "Apple iPhone 매출 폭증, Q4 예상 상회"
- "Amazon AWS 점유율 하락, MSFT Azure 추격"

SEO 규칙:
- 종목명 명시 (MSFT, Apple)
- 핵심 이슈 명확 (AI 투자, 실적, 전략)
- 구체적 수치 또는 질문 포함 ($10B, 폭증)
- 60자 이내 (모바일 표시율 고려)
```

#### 2️⃣ 10초 판독 요약 (AI 검색 최우선)

```
형식 (반드시 이 구조):

## 📌 핵심 요약 (AI 검색 엔진용)

**상황**: 1줄
  → 방금 일어난 구체적 뉴스 이벤트
  예: "Microsoft가 클라우드 인프라에 $10B 추가 투자 발표"

**영향**: 1-2줄 (수치 포함)
  → 시장에 미치는 영향
  예: "Azure 시장 점유율 23%→25% 상향 예상,
       2024년 클라우드 사업부 수익성 20% 개선"

**투자자 관점**:
  - 긍정: [구체적 이유 2개]
    예: "AI 기술 우위 확보", "엔터프라이즈 고객 확대"

  - 위험: [구체적 이유 2개]
    예: "높은 초기 투자 비용", "AWS 강력한 경쟁력"

  - 시사점: [구체적 액션]
    예: "MSFT 주가 3-6개월 강세 전망, 장기 보유 추천"

목적: AI 검색 엔진이 이 섹션만으로 뉴스 의미 파악 가능
```

#### 3️⃣ 무엇이 일어났는가 (뉴스 섹션)

```
구성 요소:

1. 명확한 헤드라인 (한 문장)
   "Microsoft, Azure 인프라에 $10B 투자 공표"

2. 언제: 정확한 날짜
   "2025년 11월 13일 발표"

3. 누가: 회사명, 경영진
   "Microsoft CEO Satya Nadella"

4. 뭐: 구체적 액션과 수치
   "Azure에 연간 $10B 투가, AI 칩 개발 가속화"

5. 출처: 공식 발표, 링크
   "Microsoft 공식 보도자료: microsoft.com/pr"
```

#### 4️⃣ 어떻게 작동하는가 (비즈니스 로직)

```
설명 내용:

1. 이 뉴스가 비즈니스에 미치는 메커니즘
   → 투자 액수 + AI 칩 개발 = 경쟁력 강화 → 고객 유입

2. 업계 배경지식 (신규 독자 고려)
   → 클라우드 시장의 AI 경쟁 상황
   → AWS vs MSFT vs Google 경쟁 구도

3. 경쟁 구도 변화
   → MSFT: AI 통합으로 기업 고객 확보
   → AWS: 규모 vs MSFT: AI 기술력

4. 시장 트렌드와의 연결
   → 엔터프라이즈 AI 도입 가속화
   → 클라우드 비용 절감 경쟁
```

#### 5️⃣ 왜 주가에 영향을 주는가 (투자 로직)

```
3가지 관점:

1️⃣ 재무 임팩트
   - 수익성 개선 메커니즘
     → AI 서비스 마진율 40% (클라우드 기본 25%)
   - 성장성 데이터
     → 클라우드 시장 연 16.4% 성장
   - 실제 수치
     → Azure 2024 매출 예상 $130B

2️⃣ 투자자 심리
   - 시장 선호도
     → 기관투자자 AI 관련 기술 회사 선호도 UP
   - 분석가 평가
     → JP Morgan, Goldman Sachs "BUY" 등급
   - 밸류에이션
     → P/E 비율: MSFT 32배 (시장평균 18배)

3️⃣ 단기 vs 장기 전망
   - 단기 (3-6개월): 주가 상승 신호
   - 중기 (6-12개월): 실적 개선 기대
   - 장기 (1-3년): 구조적 성장 추세
```

### 2.2 글쓰기 스타일 가이드

#### 글자 수
```
목표: 2,500-3,500자
- 짧음 (1,500자): SEO 약함
- 적절 (2,500-3,500자): 검색 최적화
- 너무 김 (5,000자+): 산만함
```

#### 톤
```
목표: 전문적 + 이해하기 쉬운

피할 것:
- 너무 학술적 (일반인 이해 어려움)
- 너무 캐주얼 (신뢰성 저하)
- 주관적 표현 ("반드시 상승할 것")

추천:
- "~할 것으로 예상됩니다"
- "데이터에 따르면"
- "분석가 평가는"
```

#### 단락 길이
```
규칙: 200자 이내
- AI 파싱 용이 (구조화)
- 모바일 가독성 (화면 넘김 최소화)
- 스캔 가능성 (제목→관심→읽기)
```

#### 구조
```
마크다운 계층:
# H1: 글 제목 (1개만)
## H2: 주요 섹션 (4-6개)
### H3: 소제목 (섹션별 2-3개)

리스트:
- 불릿 포인트 (항목)
1. 숫자 (순서)
```

```
마크다운 사용:
```

## 📊 Part 3: 발행 전략

### 3.1 Tier별 발행 개수

#### Tier 1: Primary (주요 - 13개 종목)

## 🎯 Part 4: SEO 최적화 전략

### 4.1 키워드 배치

```
4단계 키워드 전략:

1차 키워드 (종목명):
  - 제목에 1회
  - 첫 100자에 1회
  - 본문에 3-5회 자연스럽게
  예: MSFT, Microsoft

2차 키워드 (주제):
  - 3-5회 분산 배치
  예: "클라우드 전략", "AI 투자", "Azure"

3차 키워드 (Long-tail):
  - "~하는 이유", "~전망", "vs 비교"
  예: "MSFT AI 투자하는 이유", "MSFT vs AWS 비교"

4차 키워드 (의도):
  - "매수 기회", "투자 시점", "위험 요소"
  예: "MSFT 매수 기회", "위험 요소"

규칙: 강제 반복X, 자연스럽게 통합
```

### 4.2 AI 검색 최적화

```
AI 검색 엔진 (Claude, ChatGPT, Gemini):

우선순위:
1. 제목: 명확한 정보성
2. 핵심요약: 10초 판독 가능
3. 표: 구조화된 데이터
4. FAQ: 자연어 쿼리 매칭
5. 출처: 신뢰도

예:
사용자 질문: "MSFT AI 투자가 주가에 미치는 영향?"
AI 검색:
  → 제목 매칭
  → 핵심요약 (상황/영향)
  → 5번째 섹션 (투자 로직)
  → FAQ (주가 전망)
  → 정확한 답변 제시
```
